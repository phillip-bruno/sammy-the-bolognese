# Sammy the Bolognese 🐾

A cute photo gallery website for Sammy, the world's fluffiest Bolognese dog.

## How to Add Photos

1. Upload your photos to the `images/` folder.
2. Open `index.html` and add `<img src="images/your-photo.jpg" alt="Sammy">` inside the `#gallery` div.
3. Commit and push — the site will update automatically!
